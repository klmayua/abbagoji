# Asset Instructions

To complete the setup, replace the placeholder files in `src/assets/` with actual PNG images:

## Required Assets

### 1. icon.png (512x512px)
- Main app icon
- Should feature the GOJI2027 logo
- Format: PNG with transparency

### 2. favicon.png (48x48px)
- Web browser favicon
- Simple, recognizable at small size
- Format: PNG

### 3. splash.png (512x512px)
- App splash screen
- Should have Authority Green (#00450d) background
- GOJI2027 logo centered
- Format: PNG

### 4. adaptive-icon.png (1024x1024px)
- Android adaptive icon
- Format: PNG
- Should follow Android adaptive icon guidelines

## Asset Generation

You can generate these using:
- Figma/Sketch design exports
- Adobe Photoshop/Illustrator
- Online tools like Canva

## Current Status

The placeholder files are text files. Replace them with actual PNG images before building for production.
